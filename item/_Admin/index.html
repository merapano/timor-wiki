sub distrito

"Kecamatan" レベルである。 「郡」と訳す。 各県ごとの郡の構成については 各県のページを見よ。 【未完成】

suco

ポルトガル語で 「行政村」 ("desa")。 テトゥン語 Tetun では suku と綴られる。

Wikipidia Lista de sucos de Timor-Leste に全 Suco が掲載されている。

aldeia

"hamlet" と訳される。 物理的な家の集り、村落。

ポルトガル時代

[@mcwilliam-traube-intro] より

10の districts (conçelhos) に分かれる。 それぞれが sub districts (postos) に下位区分される。 旧「王国」は、postos の下の区分として 残った。 しかしそれらは行政的な意味しかもたなかった。 伝統的な起源の村の階層を無視し、 植民地政府は「王国」を村かなるものとして 捉えた。 「村」には Tetun 語の suku （ポルトガル語では suco である）が 使われた。
