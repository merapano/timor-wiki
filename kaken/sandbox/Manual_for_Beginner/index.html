序

このサイトは読むだけでなく、 書くことも意識して作られています。 このマニュアルは「書き方」のための マニュアルです。

既に ID と パスワードは獲得しているものとして お話しします。 login を読んでください。

三つの世界

みなさんには以下の三つの世界でそれぞれ 書き込みをしてもらいます。

  • 砂場 世界
  • 住所録 世界
  • 東ティモール百科辞典 世界

クリックすればそれぞれの世界を読めます。 このマニュアルでは「砂場世界」だけを舞台に します。

それぞれの砂場

それぞれのメンバーのために砂場を 用意してあります。

  • fukutake 福武さん用
  • ueda 上田さん用
  • morita 森田さん用
  • okuda 奥田さん用
  • nakagawa わたしの砂場

自分の砂場をクリックしてください。

最初の一歩

上から二番目の段は次のようになっています：

Edit RecentChanges Preferences ?Discussion

Edit をクリックしてください。

これで編集ができるようになりました。

書き込める部分の一番下まで行って、なにか書いてください。

１行を短めにして、 ２行か３行書いてみてください。

どんな文章でも
かまいません。
これは練習です。

そして最下段の左端の Save Page をクリックしてください。 しばらく待つと、 さきほどの文章が加えられたのが分かるでしょう。

最初の教訓は、 改行は意味がない、ということです。

次の一歩

再び Edit を押して、 編集ができる状態にしてください。

さきほどの文章に続けましょう。

今度は空行を入れてみてください。

どんな文章でも
かまいません。
これは練習です。

空行を入れてみました。
もう少し書いたら、
また空行を入れて
みましょう。

空行をはさんで次の行です。

こんどは、マウスをもって Optional comment about this change: の下の欄に カーソルをもっていき、クリックします。 これでこの欄に何か書くことができるようになりました。 ここになにかコメントを書いてください。

その上で、 Save Page をクリックしてください。

教訓は２つです：

  • 空行をはさむと段落ができる
  • Optional comment に書いたものは見えない

Comment はどうなったのでしょうか。 Edit の隣にある Recent Changes をクリックしてください。 そうです、 あなたのコメントはこのサイト全体の 「改訂履歴」にとりこまれたのです。

第三歩

こんどはリンクをつけます。 自分の砂場へ行きましょう。 Edit して次のように書いてください。

[[fukutake]]

[[ueda]]

[[morita]]

[[okuda]]

そして Save page しましょう。

すると、さきほどの追加した部分が 色が変わって表示されていることが分かるでしょう （そして [[ と ]] の部分はなくなっている はずです）。

さて、そこでその色の変わった部分をクリックしてください。

あら不思議、他の人の砂場へと行けました。

これでリンクの作成の仕方が分かりましたね。

第四歩

次は大きな一歩です。 サイト全体に影響を及ぼす作業をしてもらいます。 新しいファイルを作ります。

まずは自分の砂場へ行きましょう。 いつものように Edit から編集可能な状況に してください。

そして、 どこかに次のように書き込んでください。

[[OkudaNewFile]]

OkudaNewFile の部分は 適当な名前にしてください。 （他の人と同じ名前にならないように）

この部分がページの名前になります。 以下の注意事項を守ってください。

  • 漢字・ひらがな・カタカナは使わない
  • 半角のみ使用する
  • 記号は使わない（空白、ドル記号、円記号などなど）
  • ただし下線 _ とハイフン - だけはＯＫ
  • しかし、名前の冒頭にそれらの記号を使ってはいけない

以上の規則は厳格に守ってください。

さて、Save Page してください。 （Comment はつけてもつけなくてもけっこうです。）

さて、こんどは OkudaNewFile は表示されているのですが、 前に ? 記号がついているのが分かりますね。 これはリンクなのですが、 リンク先が存在しないリンクなのです。

？ をクリックしてください。 見慣れた Edit の画面になります。 なにか書き込みましょう。 自分の砂場へのリンクを書くのがいいでしょう。

[[okuda]]

そして Save page します。

あなたは新しいページを作成したのです。 okuda をクリックして下さい。 あなた自身の砂場に戻れたはずです。 そして、今度は OkudaNewFile の前に ？ がなくなっている筈です。 完成されたリンクとなったのです。

外へのリンク

さいごにクリックすると、 このサイトの外にでるようなリンクの作りかたを お教えします。 一例として次のようなリンクを作成しましょう。 「大阪大学」 と表示されて、 そこをクリックすると 阪大の HP にアクセスできるようなリンクです。 大阪大学の HP の URL を調べておきます。 http://www.osaka-u.ac.jp/ja です。

さて、 作りかたは簡単で： [text](url) と書けばいいだけです。 たとえば上の例ですと [大阪大学](http://www.osaka-u.ac.jp/ja) と書きます。 そうしてセーブしてそのページを見ると 大阪大学 と表示されているはずです。 「大阪大学」の部分をクリックしてください。

Voila!
