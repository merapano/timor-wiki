序

さて、中級者編です。 ここでは Markdown 記法を学びます。

このサイトの各ページ （読む側からすると「ページ」で、 書く側からすると「ファイル」と呼ぶのでしょうが 互換的に使っていきます）は、 markdown と呼ばれる記法で書かれています。

第一歩

markdown の記法はとっても簡単です。 かつて 1 多くの人が メールの範囲内で修飾をするのに 使っていた記法を そのまま採用しています。

（なんと！）M\$Word で書いたりする人の 多くなった今はすたれてきているのかもしれません。 わたしは、プレーンテキスト

修飾

斜体にしたい部分を * と * で囲みます。 *Timor Leste* と書くと Timor Leste と 表示されます。 その部分を下線（ _ と _ ）で 囲んでも同じ結果を得ることができます。 _Journal of Royal Anthropological Institute_ と 書けば、 Journal of Royal Anthropological Institute と 表示されます。

強調したい時には * を二つ使ってください。 **This is important** は This is important と表示されます。

リスト

リストの各項目を * あるいは - あるいは + で始めてください。

- First Item
- Second Item
- Third Item

と書けば、

  • First Item
  • Second Item
  • Third Item

と表示されます。 ネストも可能です。 番号をつけたい場合は 1. 2. という 番号で始めてください。2

引用、そのまま引用

引用はメールの引用と同じです。

> 先日はたいへんお世話になりました。
> 今後ともによろしく
> お願いします。

は、次のように表示されます。

先日はたいへんお世話になりました。 今後ともによろしく お願いします。

もし、改行も含めて「そのまま」 引用したい場合は、 各行の前に四つのスペースあるいは 一つのタブを置いてください。

章、節

HTML の用語「ヘッダレベル」を使って 説明します。 それぞれの章・節のタイトルの前に # の記号を決められた数だけ置けば、 章・節が表現できます。

# 章のタイトル
## 節のタイトル1
### 小節のタイトル1
### 小節のタイトル2
## 節のタイトル2
### 小節のタイトル3

「全体のタイトル」という考え方はありません。 ですから、 「章のタイトル」を「全体のタイトル」と 考えたほうがいいかもしれません。 （そして「章」は「節」へ、 「節」は「小節」へと読み替えてください。）

なお

 # なんやらかんやら

 ## なんとかかんとか

を

 なんやらかんやら
 ================

 なんとかかんとか
 ----------------

と書くこともできます。 まったく互換的ですので、 好きなほうを使ってください。

うまくいかない場合

砂場で実験をして、 「どうもうまくいかない」という場合は、 スペースを見てください。

  • リストの場合、
    • - と 本文の間にスペース
    • 1. と本文の間にスペースを入れます
  • 文字修飾の場合、
    • *文字* の前後にスペース
    • _文字_ の前後にスペースを入れます
  • 章・節の場合、
    • # とタイトルの間にスペースを入れます
  • 引用の場合
    • > と本文の間にスペースを入れます

これらのスペースを入れるのを忘れてませんか？ あるいは、全角のスペースをいれていませんか？ そうならば、 全角のスペースを半角のスペースに修正してください。

拡張

いくつか特別な記法があります。

[[!toc ]]

と書けば TOC、すなわち Table of Contents （目次）が その場所に書かれます。

脚注は [^1] という記号 （1 の部分は何でもけっこうです。 ただし、全て異なる数字を入れてください） を脚注をつけたい所に書きます。

つづいて、脚注をつけた段落の最後まで行き、 １行あけて次のように書きます：

\[^1]: これが脚注です。
   2行目以降は
   インデントをします

これで脚注のできあがりです。
