あたらしいページの名前

  • 漢字・ひらがな・カタカナは使わない
  • 半角のみ使用する
  • 記号は使わない（空白、ドル記号、円記号などなど）
  • ただし下線 _ とハイフン - だけはＯＫ
  • しかし、名前の冒頭にそれらの記号を使ってはいけない

大文字、小文字は慣例 （固有名詞は大文字などなど）に したがってください。 （無闇に大文字を使用しないように） ハイフンも同様です。

下線は空白の代わりに使用します。

以下に例を示します。

 [[Merlau-Ponty]] 
 [[UNTL]] 
 [[trade]] 
 [[this_is_a_long_title_for_a_link]]

各世界の独立

砂場、 住所録、 百科事典は 独立しています。 相互リンクはできません。1 住所録の _Admin と 百科事典の _Admin は違うものです。

知っていて、 コンピュータの知識があれば可能ですが、 なるべく避けてください。 とりわけ公開（予定）の「百科事典」から 非公開の「住所録」へのリンクは避けてください。

百科事典の下位世界

「百科事典」の世界には２つの下位世界があります。2 「文献」小世界と「フィールドワーク」小世界とです。

  • 「文献」小世界
  • 「フィールドワーク」小世界

文献小世界

文献小世界にはすでに1000ほどの文献が用意されています。 「百科事典」の世界の入口 から 目次で「文献」をクリックして、 さらに_Bib をクリックしてください。 キーがリストになって表示されます。

著者の名前でキーが作られ、 リストはアルファベット順に並んでいます。 （ただし、編著の場合は名前の前に z. を つけています。

クリックすれば、 そのキーの示す文献の詳細が表示されます。

フィールドワーク小世界

「百科事典」の世界の入口 から 目次で「フィールドワーク」をクリックして、 さらに_FW の部分をクリックしてください。 キーがリストになって表示されます。

リンクの貼り方

それぞれの小世界のページに、 親世界である「百科事典」からリンクを貼るには、 キーの前に (1)「文献」の場合には bib/、 (2)「フィールドワーク」の場合には fw/ を つけてください。

  • 「文献」小世界 bib/
  • 「フィールドワーク」小世界 fw/

[[bib/fox-tracing]] とすれば、 fox-tracing をキーとする文献 Tracing the Path, Recounting the Past へとリンクが貼られます。 （上のリンクをクリックしてください。）

あるいは [[fw/2013-08-28-ub-01]] と書けば、 2013-08-28-ub-01 をキーとする フィールドワークのノーツにリンクを貼ることができます。

作成の仕方

文献小世界

用意された文献以外を載せたい場合は、 わたしにメールで詳細を伝えてください。 わたしが作成した上で、 メーリングリストでキー名をお伝えします。

ある文献にコメントやサマリーなどをつけくわえたいときは、 fox-tracing ではなく （これは編集は禁止します 3）、 そのページの最後にリンクのある -summary の ついたページ （fox-tracing-summary） を編集してください。

フィールドワーク小世界

こちらはどんどん作成してください。 見栄え、あとからの利用を考えて、 キー（ページのタイトル、名前）は 2013-08-28-ub-001 のようにしたいと思います。 2013-08-28 は日付、 ub はインフォーマント、 場所等々への符丁です。 その上で、 001 から始まる番号をつけて区別していきましょう。

フィールドワーク小世界から親世界への リンクは、 親世界の中でやっているのと 同じように作ってかまいません （fw/ をつける等の必要はありません）。

なるべく（公開予定である） 「百科事典」から「フィールドワーク小世界」への リンクは避けてください。

おまけ：ディレクトリ構成

以上は、 わかりやすい様に比喩を多様しました。 じっさいのコンピュータのなかみ （ディレクトリ（フォルダ）構成）は つぎのようになっています。 こちらのほうが分かりやすい方は、 こちらを見ながら、 上の説明を読み直してください。

  timor-web/ ----- file1
              |
              +--- file2
              |
              +--- file3
              |
              +--- kaken/ ----- files
              |             |
              |             +-- sandbox/
              |             |
              |             +--- contact/ (1)
              |
              +--- literature/
              |
              +--- media/
              |
              +--- item/ -------- files (2)
                            |
                            +---- bib/
                            |
                            +---- fw/

(1) が「住所録」世界、 (2) が「百科事典」世界です。

timor-web 直下のページには だれでもアクセスできます （読むことができます）。

kaken/ literature/ media/ item へは 科研メンバーとして ログインをする必要があります。

ファイルを編集するには、 登録ユーザーとしてログインする必要があります。

  1. ディレクトリ（フォルダ）構造を↩
  2. 下位ディレクトリがある、ということです。↩
  3. 文献のページはマシンで作成しています。 すなわち、あなたが編集したページは 上書きされてしまうのです。↩
